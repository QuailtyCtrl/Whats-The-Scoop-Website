# What's The Scoop?

Website for What's The Scoop — Innisfil's neighbourhood ice cream shop at 1070 Innisfil Beach Road.

Built with [TanStack Start](https://tanstack.com/start), React 19, Tailwind CSS v4, deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build   # produces .vercel/output/
```

Push to `main` → Vercel auto-deploys via GitHub integration.
